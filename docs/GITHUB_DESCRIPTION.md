RetailPulse 360 is an Executive Customer Churn Intelligence Platform developed using SQL, Python, R, Power BI and Streamlit.

The project performs customer churn analysis through automated ETL pipelines, statistical analysis, interactive dashboards and executive-level KPI reporting.

Key Features

• Automated Data Cleaning
• Feature Engineering
• SQL Analytics
• Customer Lifetime Value
• Revenue at Risk Analysis
• Statistical Testing using R
• Interactive Streamlit Dashboard
• Executive Power BI Dashboard
